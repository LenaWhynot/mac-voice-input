# 🎤 Голосовой ввод на Mac — бесплатно, локально

Вводите текст голосом в любое поле на маке — заметки, мессенджеры, браузер, редактор кода. Распознавание идёт **локально** через Whisper: аудио никуда не уходит, подписки не нужны.

Заменяет Superwhisper (~$8.49/мес) и Wispr Flow (~$12/мес) — **бесплатно после установки**.

> Только для **Apple Silicon** (Mac M1/M2/M3/M4).

## Установка

1. Нажмите зелёную кнопку **Code → Download ZIP** (или `git clone` этот репозиторий) и распакуйте.
2. Двойной клик по **`install.command`**.
   *Если macOS пишет «неизвестный разработчик» — правый клик по файлу → «Открыть» → «Открыть».*
3. Установщик всё поставит сам и в конце откроет две панели настроек. Поставьте галочку Python в обеих (путь уже в буфере — в окне выбора `Cmd+Shift+G`, вставьте, Enter, Open, включите тумблер):
   - **Универсальный доступ** (Accessibility)
   - **Мониторинг ввода** (Input Monitoring)

Готово.

## Как пользоваться

- **Тык правого Option** → говорите → **ещё тык** → текст печатается в активное поле.
- Работает в любом приложении. Запускается сама при включении мака.

## Первый запуск

Первый голосовой ввод скачает модель (~1.6 ГБ). Если вы в России и скачивание упирается в лимит HuggingFace — включите VPN на момент первого запуска или добавьте бесплатный токен HuggingFace.

## Если залипло

```bash
launchctl bootout gui/$(id -u)/com.whisper.dictation
launchctl bootstrap gui/$(id -u) ~/Library/LaunchAgents/com.whisper.dictation.plist
```

## Удалить

```bash
launchctl bootout gui/$(id -u)/com.whisper.dictation
rm ~/Library/LaunchAgents/com.whisper.dictation.plist
rm -rf ~/whisper-dictation ~/.config/whisper-skill
```

## Что внутри

Установщик скачивает оригинальный движок **[Whisper-Skill](https://github.com/Mobiss11/Whisper-Skill)** (автор [@Mobiss11](https://github.com/Mobiss11), все права на движок принадлежат автору) и накатывает поверх набор улучшений под macOS (`improvements.patch`): надёжная вставка текста прямой печатью, toggle по правому Option, звуковые сигналы, авто-стоп записи, отключение прогрева, ломавшего захват клавиш. Этот репозиторий **не хранит** чужой код — только установщик и патч.
