# 🎤 Whispee — голосовой ввод на Mac, бесплатно и локально

Вводите текст голосом в любое поле на маке — заметки, мессенджеры, браузер, редактор кода. Распознавание идёт **локально** через Whisper: аудио никуда не уходит, подписки не нужны.

Заменяет Superwhisper (~$8.49/мес) и Wispr Flow (~$12/мес) — **бесплатно после установки**.

> Только для **Apple Silicon** (Mac M1/M2/M3/M4).

## Установка

1. **Code → Download ZIP** (или `git clone`) и распакуйте.
2. Двойной клик по **`install.command`**.
   *Если macOS пишет «неизвестный разработчик» — правый клик по файлу → «Открыть» → «Открыть».*
3. В конце установщик откроет две панели настроек. Поставьте галочку **Whispee** в обеих (путь уже в буфере — `Cmd+Shift+G`, вставьте, Enter, Open, включите тумблер):
   - **Универсальный доступ** (Accessibility)
   - **Мониторинг ввода** (Input Monitoring)

Готово.

## Как пользоваться

- **Тык правого Option** → говорите → **ещё тык** → текст вставляется в активное поле.
- Или запускайте запись из меню-бара (иконка 🎤 → «Начать / остановить запись»).
- Работает в любом приложении, запускается само при включении мака.

## Меню-бар 🎤

Иконка в верхней панели — быстрое управление без терминала:
- **Начать / остановить запись** — активировать микрофон из меню
- **Перезапустить** — если подвисло
- **Полностью выключить / Включить**
- **Открыть лог**

## Что внутри

- Движок: **[Whisper-Skill](https://github.com/Mobiss11/Whisper-Skill)** (автор [@Mobiss11](https://github.com/Mobiss11), все права на движок — у автора). Установщик скачивает его напрямую у автора и накатывает улучшения под macOS (`improvements.patch`): toggle по правому Option, авто-вставка, звуки, авто-стоп, авто-сброс зависшей транскрипции, запуск из меню-бара.
- Модель: **large-v3-turbo** (Apple Metal, ~0.3–0.5с на фразу). Скачивается с **ModelScope** — без лимитов HuggingFace, поэтому работает и из России без VPN.
- Приложение **Whispee.app** с иконкой (имя и логотип в системе, а не «Python»).

Этот репозиторий **не хранит** чужой код — только установщик, патч и иконки.

## Удалить

```bash
launchctl bootout gui/$(id -u)/com.whisper.dictation
launchctl bootout gui/$(id -u)/com.whispee.control
rm ~/Library/LaunchAgents/com.whisper.dictation.plist ~/Library/LaunchAgents/com.whispee.control.plist
rm -rf ~/whisper-dictation ~/.config/whisper-skill ~/Applications/Whispee.app
```
